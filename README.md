## 📌 Project Overview

**Ganesh Cyber Shield** is a web-based cybersecurity toolkit designed to perform essential security analysis tasks such as vulnerability scanning, port scanning, link crawling, and directory enumeration.

This project demonstrates practical implementation of cybersecurity concepts using Python and web technologies, making it a valuable tool for beginners and security enthusiasts.

---

## 🎯 Features

🔍 **Vulnerability Scanner**
- Detects basic SQL Injection and XSS vulnerabilities
- Uses payload-based testing approach

🌐 **Link Crawler**
- Extracts all hyperlinks from a target website
- Useful for reconnaissance and web structure analysis

📁 **Directory Scanner**
- Identifies hidden or sensitive directories
- Uses wordlist-based brute force approach

🔓 **Port Scanner**
- Scans common ports (21, 22, 80, 443, etc.)
- Detects open services on target systems

💻 **Network Information**
- Displays system hostname and local IP address

## 🛠️ Installation & Setup (Local)

1. Clone the repository:


git clone [https://github.com/ganesh94jrl/Cyber_Security_Toolkit-.git](https://github.com/ganesh94jrl/Cyber_Security_Toolkit-.git)



2. Navigate to project folder:


cd Cyber_Security_Toolkit-



3. Install dependencies:


pip install -r requirements.txt



4. Run the application:

python app.py



5. Open in browser:


[http://127.0.0.1:5000](http://127.0.0.1:5000)
